這是一份名為「**AI 魔法之旅：從零基礎到網站發佈通關手冊**」的綜合教學指南。

### 核心內容摘要

這份手冊專為**零程式基礎**、使用**一般文書電腦**（Win11、無高階顯卡）的初學者設計，旨在透過友善且循序漸進的方式，引導讀者掌握 Claude AI 工具，並最終完成個人網站的製作與發佈。

手冊內容分為四大章節：

1.  **第一章：準備好你的裝備**
    *   強調只需最基本的硬體（能上網的電腦）與瀏覽器。
    *   介紹三位 AI 夥伴：**Claude**（萬能秘書）、**Claude Code**（工程師模式）、**Claude Skills**（技能包）。

2.  **第二章：學會跟 AI 說話（提示工程）**
    *   傳授「**角色 + 目標 + 限制 + 輸出格式**」的萬能提示公式。
    *   提供 20 條隨查隨用的「提示寶箱」模板，涵蓋理解概念、產生程式碼、除錯優化及文書整理等場景。

3.  **第三章：實戰演練——城市與旅行**
    *   **專案一：資料整理小管家**：示範如何將雜亂的城市筆記轉換為結構化的表格資料，並進行排序與延伸生成。
    *   **專案二：旅行推薦小助理**：模擬產品經理思維，引導 AI 設計具備景點推薦、預算分級與標籤功能的旅遊規劃應用。

4.  **第四章：讓全世界看見——網站發佈教學**
    *   提供 GitHub Pages 的完整發佈流程（懶人包）：
        *   **建立新家**：在 GitHub 建立 Public Repository。
        *   **準備檔案**：解壓縮 `deploy-package.zip`。
        *   **上傳檔案**：將 `index.html` 與 `assets` 資料夾上傳至 GitHub。
        *   **開啟功能**：設定 GitHub Pages 的 Branch 為 `main` 並儲存。
        *   **獲得網址**：等待部署完成，取得專屬網站連結。

附錄部分整理了關於英文能力、錯誤處理及備份還原（使用 `source-code.zip`）的常見問題，並鼓勵讀者保持好奇心，運用所學技能繼續創作。
